# 服务器和大厅

csgo_gc 包含客户端和专用服务器支持。它使用 Steam P2P 接口进行网络通信，并实现了足够的 GC 行为来支持大厅、服务器浏览器可见性、SO cache 转发，以及部分和游戏物品状态相关的行为。

## 大厅

当前项目报告大厅功能可用。这并不代表实现了匹配系统。大厅和匹配是不同概念：

- 大厅用于协调玩家。
- 匹配需要中心化服务，因此不在计划内。

## 专用服务器

项目包含专用服务器支持。发布包可能会替换 `srcds.exe` 或平台对应的服务器启动器文件，安装前请先备份。

## 服务器浏览器

默认情况下，csgo_gc 可以让服务器浏览器只显示带有 csgo_gc 标记的服务器：

```text
"show_csgo_gc_servers_only" "1"
```

只有在你明确想查看更广泛的旧版 CS:GO 服务器列表时，才建议关闭它。

## 服务器上的物品状态

ServerGC 路径会验证并转发部分客户端 SO cache 消息。它还支持通过客户端到服务器路径传播 StatTrak 音乐盒 MVP 计数状态。

这是本项目的本地行为，不是 Valve 官方后端行为。
