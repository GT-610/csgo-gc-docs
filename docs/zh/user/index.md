# 用户指南

本节面向希望通过 csgo_gc 恢复旧版 CS:GO 本地 GC 功能的玩家和服务器管理员。

## 推荐阅读路径

1. 修改任何游戏文件前，先阅读[安装](/zh/user/installation)。
2. 查看[配置](/zh/user/configuration)，尤其是 `appid_override`、日志和 RCON 设置。
3. 通过[库存](/zh/user/inventory)了解库存数据的工作方式。
4. 只有在需要实时物品变更或外部工具接入时，再使用 [RCON](/zh/user/rcon)。
5. 如果计划搭建或加入 csgo_gc 服务器，请阅读[服务器和大厅](/zh/user/servers)。

## 重要提醒

发布包会替换 `csgo.exe`、`srcds.exe` 或平台对应的启动器可执行文件。安装前请先备份原始文件。
